# LLMCrowdPackets 实验模拟系统

## 项目简介

本项目通过模拟攻击者与防御系统的对抗过程，研究大语言模型（LLM）在非法内容生成场景下的行为特征及防御策略效果。支持多维度策略配置，包含攻击者行为建模、防御策略体系、成本评估模型三大核心模块。

## 目录结构

```bash
LLMCrowdPackets/ 
├── assets/ # 配置参数模块 
│ └── parameters.py # 模型参数及辅助函数 
├── utils/ # 工具模块 
│ ├── defender.py # 防御策略实现 
│ └── question.py # 问题对象定义 
├── main.py # 主程序入口 
└── 7.14.md # 实验设计文档（核心规范）
```

## 使用

运行``main.py``即可
